jsDelivrзҡ„жӯЈзЎ®жү“ејҖж–№ејҸ

Created2020-02-08|Updated2020-04-18|Software
|Word count:970|Reading time:3min|Post View:|Comments:

ж–Үз« е·Із»ҸеңЁ2020.4.5жӣҙж–°

ејҖе§ӢжҢҒз»ӯй«ҳдә§еүҚеҮ еӨ©ејҖе§ӢпјҢgithubзҡ„rawж–Үд»¶дёӢиҪҪеҹҹеҗҚraw.githubusercontent.comиў«еўҷдәҶпјҢеҜјиҮҙжҲ‘зҡ„зҪ‘з«ҷеҫҲеӨҡеӣҫзүҮйғҪжҳҜ404пјҲеӣ дёәжҲ‘жҳҜзӣҙжҺҘдҪҝз”Ёgithubзҡ„ж–Үд»¶пјүпјҢжҲ‘иҪ¬дёәдҪҝз”Ёcloudflareзҡ„workersеҸҚд»ЈгҖӮдҪҶжҳҜеҸҚд»ЈжңүжҜҸж—Ҙ10Wж¬Ўзҡ„иҜ·жұӮж¬Ўж•°йҷҗеҲ¶гҖӮдёҮдёҖд»ҘеҗҺжҲ‘зҡ„зҪ‘з«ҷи®ҝй—®йҮҸеўһеӨ§дәҶе‘ўпјҹиҝҷж ·еІӮдёҚжҳҜдёҚеӨҹз”ЁпјҹпјҲеңЁжғіPeachпјү

д»ҠеӨ©ж—©дёҠжҲ‘жүҚеңЁгҖҗж—Ҙеёёеҗҗж§Ҫ04гҖ‘зҡ„иҜ„и®әеҢәйҮҢйқўиҜҙдёҚдјҡз”ЁjsDelivrпјҢеҲ°дәҶжҷҡдёҠпјҢе—ҜпјҢзңҹйҰҷвҖҰвҖҰ

jsDelivrжҳҜдёҖдёӘжҜ”иҫғеҘҪзҡ„CDNе№іеҸ°пјҢе®ҳж–№еҸ·з§°jsDelivr вҖ“ Open Source CDN free, fast, and reliableпјҢз®ҖеҚ•жқҘиҜҙе°ұжҳҜејҖжәҗзҡ„CDNпјҢе…Қиҙ№гҖҒеҝ«гҖҒеҸҜйқ иҝҷж ·зҡ„

дёҚиҝҮзЎ®е®һпјҢиҝҷзҺ©ж„Ҹзҡ„еҸЈзў‘д№ҹжҢәеҘҪпјҢйӮЈжҲ‘е°ұжҢүз…§жҲ‘еҚҠеӨ©зҡ„дҪҝз”ЁдҪ“йӘҢпјҢжқҘиҜҙиҜҙиҝҷзҺ©ж„Ҹзҡ„жӯЈзЎ®жү“ејҖж–№ејҸеҗ§

дҪ йңҖиҰҒеҮҶеӨҮпјҡ

дёҖдёӘgithubиҙҰеҸ·

ејҖе§Ӣж“ҚдҪң

дҪ йңҖиҰҒзҷ»еҪ•дҪ зҡ„GithubпјҢеҲӣе»әдёҖдёӘдҪ жғіз”ЁжқҘж”ҫж–Үд»¶зҡ„д»“еә“пјҢз„¶еҗҺеңЁиҝҷдёӘд»“еә“йҮҢйқўдёҠдј дҪ зҡ„ж–Үд»¶пјҢеғҸжҲ‘иҝҷж ·

з„¶еҗҺзӮ№еҮ»ReleaseпјҢж–°е»әдёҖдёӘзүҲжң¬пјҢеңЁдёҠйқўзҡ„е°Ҹж–№жЎҶйҮҢйқўеЎ«еҶҷдҪ зҡ„зүҲжң¬еҸ·пјҢе°ҪйҮҸеЎ«еҶҷж•°еӯ—пјҢдҫӢеҰӮ1.0д№Ӣзұ»зҡ„пјҢдёҚиҰҒз”Ёдёӯж–ҮпјҒпјҒпјҒ

жҺҘзқҖзӣҙжҺҘи°ғз”ЁjsDelivrпјҢдҫӢеҰӮжҲ‘еңЁеҗҚдёәPicture-repoзҡ„д»“еә“еҸ‘еёғдәҶ1.0зүҲжң¬пјҢйӮЈд№ҲжҲ‘и®ҝй—®й“ҫжҺҘпјҡ

https://cdn.jsdelivr.net/gh/GamerNoTitle/Picture-repo@1.0/AboutMe/logo-mini.png

е°ұеҸҜд»ҘзӣҙжҺҘи°ғз”ЁжҲ‘зҡ„еӨҙеғҸпјҢжҢүз…§е®ҳж–№зҡ„ж јејҸпјҢе°ұжҳҜ

https://cdn.jsdelivr.net/gh/<username>/<repo-name>@<version>/<path>

зҡ„ж ·еӯҗпјҢи§ЈйҮҠдёҖдёӢпјҡ

<username>е°ұж”№жҲҗиҮӘе·ұзҡ„еҗҚеӯ—пјҢ<repo-name>ж”№жҲҗиҮӘе·ұзҡ„д»“еә“еҗҚеӯ—пјҢ<version>е°ұжҳҜдҪ зҡ„releaseзүҲжң¬пјҢеҰӮжһңдёҚеЎ«дјҡиҮӘеҠЁйҖүжӢ©жңҖж–°зҡ„release<path>ж”№жҲҗиҮӘе·ұзҡ„ж–Үд»¶и·Ҝеҫ„гҖӮеҪ“жҲ‘дёҠдј зҡ„ж—¶еҖҷжҲ‘зҡ„ж–Үд»¶еӨ№и·Ҝеҫ„жҳҜ.\folder\example.pngзҡ„иҜқпјҢйӮЈд№Ҳ<path>е°ұиҰҒж”№жҲҗfolder/example.png

дҪҶжҳҜиҜ·жіЁж„ҸпјҒеҪ“дҪ зҡ„releaseеҢ…еӨ§дәҺ50MBпјҢйӮЈд№Ҳjsdelivrдјҡз»ҷдҪ жҠҘй”ҷе№¶дё”дёҚз»ҷдҪ жҸҗдҫӣеҠ йҖҹжңҚеҠЎпјҢдҫӢеҰӮдёӢйқўиҝҷжқЎй“ҫжҺҘ:

https://cdn.jsdelivr.net/gh/NotFoundNEKKO/Storage@1.0/иЎЁжғ…еҢ…/зңҹеҸ«дәәиҙЁеЈҒеҲҶзҰ».jpg

зӮ№ејҖе°ұдјҡеҸ‘зҺ°жҸҗзӨәпјҡ

Package size exceeded the configured limit of 50 MB. Try https://github.com/NotFoundNEKKO/Storage/tree/1.0/иЎЁжғ…еҢ…/зңҹеҸ«дәәиҙЁеЈҒеҲҶзҰ».jpg instead.

жүҖд»ҘиҜҙиҝҳжҳҜиҰҒе°ҪйҮҸеҮҸе°‘иҮӘе·ұжҜҸдёӘreleaseзҡ„еӨ§е°ҸпјҢеҰӮжһңиҜҙеӨӘеӨ§дәҶе»әи®®еҲҶжҲҗеҮ дёӘд»“еә“ж”ҫе“Ұ~

д»ҺзҫӨеҸӢйӮЈйҮҢеӯҰжқҘзҡ„пјҡеҪ“дҪ зҡ„зүҲжң¬еҸ·еҶҷдёәmasterж—¶пјҢеҸӘйңҖиҰҒз¬¬дёҖж¬ЎеҸ‘еёғreleaseеҚіеҸҜпјҢеҗҺйқўзӣҙжҺҘз”ЁmasterеҲҶж”Ҝзҡ„ж–Үд»¶пјҢд№ҹжІЎжңү50MBзҡ„ж–Үд»¶еҢ…еӨ§е°ҸйҷҗеҲ¶

2020.4.5 жӣҙж–°пјҡGithub Pageжі•

ж №жҚ®иҜ„и®әеҢә@Anonymousзҡ„еӣһеӨҚпјҢеҸ‘зҺ°жңүеҸҰеӨ–дёҖз§Қж–№ејҸд№ҹиғҪеӨҹи®ҝй—®Githubдёӯзҡ„ж–Үд»¶пјҲиҖҢдё”иҝҳжҳҜrawпјү

жҲ‘д»¬иҝҳжҳҜд»ҘдёҠйқўйӮЈдёӘд»“еә“дёәдҫӢеӯҗпјҢе…ҲеҲ°д»“еә“дёӯзҡ„и®ҫзҪ®пјҢжү“ејҖGithub PagesжңҚеҠЎ

з„¶еҗҺжҲ‘д»¬еҸҜд»ҘзңӢеҲ°Githubе°ҶжҲ‘зҡ„иҝҷдёӘд»“еә“зҡ„pageжңҚеҠЎйғЁзҪІеҲ°дәҶhttps://bili33.top/Picture-repo/иҝҷдёӘй“ҫжҺҘ

жҲ‘д»¬е°қиҜ•и®ҝй—®д»“еә“дёӯзҡ„ж–Үд»¶

еңЁиҝҷйҮҢжҲ‘и®ҝй—®зҡ„жҳҜжҲ‘зҡ„зҪ‘з«ҷеӨҙеӣҫвҶ“

еҸҜд»ҘзңӢеҲ°пјҢзҡ„зЎ®жҳҜеҸҜд»ҘиҝӣиЎҢи®ҝй—®зҡ„гҖӮиҖҢеҰӮжһңеңЁGithub PagesжңҚеҠЎдёӯејҖеҗҜиҮӘе®ҡд№үеҹҹеҗҚзҡ„иҜқпјҢиҝҳиғҪеӨҹй…ҚеҗҲCloudFlareзҡ„CDNиҝӣиЎҢдёҖжіўеҠ пјҲjianпјүйҖҹ

йўҳеӨ–иҜқпјҡ

дҪҶжҳҜиҝҳжҳҜжңүдәәиҰҒзӣҙжҺҘи®ҝй—®raw.githubusercontent.comеҹҹеҗҚпјҢжүҖд»ҘжҲ‘еңЁиҝҷйҮҢејҖж”ҫеҮәжҲ‘зҡ„еҸҚд»Ј

еҪ“дҪ и®ҝй—®дәҶеҹҹеҗҚдёәraw.githubusercontent.comзҡ„ж–Үд»¶еҗҺпјҲеҗҺйқўеә”иҜҘжңүдёҖеӨ§дёІзҡ„ж–Үд»¶и·Ҝеҫ„й“ҫжҺҘпјүпјҢжҠҠе®ғж”№жҲҗдёӢйқўд»»ж„ҸдёҖдёӘеҹҹеҗҚ

еҸҚд»Ј1пјҡcdn.bili33.topпјҲжҜҸж—ҘиҜ·жұӮдёҠйҷҗиҫғе°‘пјҢеӣ дёәжҲ‘зҡ„еҸҚд»ЈйғҪеңЁиҝҷйҮҢпјү

еҸҚд»Ј2пјҡbilicdn.tkпјҲжҺЁиҚҗпјү
